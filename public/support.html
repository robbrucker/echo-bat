Common things

Tilt isn't working on iPhone. Open Settings → Privacy & Security → Motion & Fitness and make sure both the system toggle and Echo Bat's row are on. Touch zones at the top and bottom of the screen always work as a backup — they don't need any permission.

The bat feels too sensitive / not sensitive enough. The neutral pose is whatever you're holding the phone like at the moment a run starts; reset it by crashing and starting over.

Sound is muted. The iOS app is set to play even with the silent switch on, but if you've muted the volume entirely or hit Do Not Disturb that takes priority. Web Safari respects whatever your browser tab's mute state is.

I died at the last possible second and want my distance back. Sorry. The cave is implacable. Try a fresh ping.

Bug reports

Email helps the most when it includes:

  • What you were doing when it happened
  • Phone model + iOS version, or browser + OS for the web version
  • A screenshot if it was a visual bug
